Syokai 完全オフラインPWA版

GitHubのSyokaiフォルダに、この中身をそのまま上書きしてください。

含まれるファイル:
- index.html
- manifest.json
- sw.js
- icon-192.png
- icon-512.png

使い方:
1. GitHub Pagesにアップロード
2. iPadのSafariで一度オンライン状態で開く
3. 画面を一通り表示して画像を読み込む
4. 共有 → ホーム画面に追加
5. 以後、ホーム画面の「Syokai」から起動

注意:
初回だけオンラインが必要です。初回表示後は、Service WorkerがHTMLと画像URLをキャッシュします。
画像数: 38件
